# 2025AI_Giraffe
2025년 AI 프로젝트를 위한 레포지토리 - 기린과 사람 얼굴 합성 프로그램

## Background Removal

1. Install dependencies:
   ```bash
   pip install rembg pillow
   ```
2. Run the script to clean images:
   ```bash
   python remove_background.py
   ```
   This reads images from `after_prep_img` and writes the cleaned versions to `after_prep_img_clean`.
